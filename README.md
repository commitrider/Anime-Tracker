# Vue Anime Tracker
Simple anime tracker made with Vue and Jinkan API.

<br/>

## :hammer_and_pick: Built With
* Vue
* Vite
* JavaScript
* CSS
* [Jinkan API](https://jikan.moe/)

<br/>

## Project setup
```
npm install
```

### Compiles for development
```
npm run dev
```
